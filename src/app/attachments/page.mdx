# 开发者快速入门

## 使用 OpenAI API 快速入门

OpenAI API 提供了一个简单的界面，让开发者能够在其应用中创建一个智能层，由 OpenAI 的最新模型驱动。Chat Completions 端点是 ChatGPT 的基础，提供了一种简单的方法来使用类似 GPT-4 的模型将文本作为输入并生成输出。

[查看 API 参考](/docs/api-reference)

如果您是一名经验丰富的开发者或希望直接了解如何使用 OpenAI API，那么 API 参考或文本生成指南是开始的好地方。在本快速入门中，您将学习：

- 如何设置您的开发环境
- 如何安装最新的 SDK
- OpenAI API 的一些基本概念
- 如何发送您的第一个 API 请求

如果您在开始使用 OpenAI API 时遇到任何问题或有任何疑问，请加入我们的[开发者社区](https://community.openai.com)。

Python 是一种流行的编程语言，常用于数据应用、Web 开发和许多其他编程任务，因为它易于使用。OpenAI 提供了一个[定制的 Python 库](https://github.com/openai/openai-python)，使在 Python 中使用 OpenAI API 变得简单高效。

## 步骤 1：设置 Python

### 安装 Python

要使用 OpenAI Python 库，您需要确保已安装 Python。一些计算机预先安装了 Python，而其他计算机则需要您自己设置。要测试是否已安装 Python，请导航到终端或命令行：

- MacOS：打开终端：您可以在应用程序文件夹中找到它，或使用 Spotlight（Command + Space）进行搜索。
- Windows：打开命令提示符：您可以在启动菜单中搜索“cmd”。

然后，输入单词 `python`，然后按回车/enter。如果您进入了 Python 解释器，那么您的计算机上已安装了 Python，可以进行下一步。如果您收到类似于“Error: command python not found”的错误消息，那么您可能需要安装 Python 并使其在终端/命令行中可用。

要下载 Python，请访问 [官方 Python 网站](https://www.python.org/downloads/) 并下载最新版本。要使用 OpenAI Python 库，您需要至少安装 Python 3.7.1 或更高版本。如果您是第一次安装 Python，可以遵循 [官方 Python 安装指南](https://wiki.python.org/moin/BeginnersGuide/Download) 进行操作。

### 设置虚拟环境（可选）

一旦安装了 Python，创建虚拟 Python 环境来安装 OpenAI Python 库是一个好习惯。虚拟环境提供了一个干净的工作空间，用于安装 Python 包，以避免与其他项目中安装的库发生冲突。如果您正在第一次使用 Python，可以遵循 [官方 Python 虚拟环境指南](https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/) 进行操作。

要创建虚拟环境，Python 提供了一个内置的 `venv` 模块，它提供了虚拟环境所需的基本功能。运行以下命令将在当前文件夹中创建一个名为“openai-env”的虚拟环境：

```bash
python -m venv openai-env
```

创建虚拟环境后，需要激活它。在 Windows 上，运行：

```bash
openai-env\Scripts\activate
```

在 Unix 或 MacOS 上，运行：

```bash
source openai-env/bin/activate
```

您应该看到终端/命令行界面发生了轻微的变化，现在左侧显示“openai-env”。有关使用虚拟环境的更多详细信息，请参阅 [官方 Python 文档](https://docs.python.org/3/tutorial/venv.html#creating-virtual-environments)。

### 安装 OpenAI Python 库

一旦安装了 Python 3.7.1 或更高版本，并且（可选）设置了虚拟环境，就可以安装 OpenAI Python 库了。从终端/命令行运行：

```bash
pip install --upgrade openai
```

完成后，运行 `pip list` 将显示您在当前环境中安装的 Python 库，其中应该包含 OpenAI Python 库。

## 步骤 2：设置您的 API 密钥

### 设置您的 API 密钥以供所有项目使用（推荐）

将 API 密钥设置为所有项目可访问的主要优势是 Python 库会自动检测并使用它，而无需编写任何代码。

#### MacOS

1. **打开终端**：您可以在应用程序文件夹中找到它，或使用 Spotlight（Command + Space）进行搜索。
2. **编辑 Bash 配置文件**：使用命令 `nano ~/.bash_profile` 或 `nano ~/.zshrc`（对于较新的 MacOS 版本）打开配置文件进行编辑。
3. **添加环境变量**：在编辑器中，添加以下行，将 `your-api-key-here` 替换为实际的 API 密钥：

```bash
export OPENAI_API_KEY='your-api-key-here'
```

4. **保存并退出**：按 Ctrl+O 写入更改，然后按 Ctrl+X 关闭编辑器。
5. **加载您的配置文件**：使用命令 `source ~/.bash_profile` 或 `source ~/.zshrc` 加载更新后的配置文件。
6. **验证**：在终端中键入 `echo $OPENAI_API_KEY` 应该显示您的 API 密钥。

#### Windows

1. **打开命令提示符**：您可以在启动菜单中搜索“cmd”。
2. **设置环境变量**：在当前会话中，使用以下命令设置环境变量，将 `your-api-key-here` 替换为实际的 API 密钥：

```bash
setx OPENAI_API_KEY "your-api-key-here"
```

3. **永久设置**：要使设置永久生效，请通过系统属性将变量添加到系统环境变量中：
	1. 右键单击“此电脑”或“我的电脑”，然后选择“属性”。
	2. 单击“高级系统设置”。
	3. 单击“环境变量”按钮。
	4. 在“系统变量”部分中，单击“新建...”，然后将 `OPENAI_API_KEY` 作为变量名称，您的 API 密钥作为变量值。
4. **验证**：重新打开命令提示符，然后键入 `echo %OPENAI_API_KEY%` 应该显示您的 API 密钥。

### 为单个项目设置您的 API 密钥

如果您只希望 API 密钥对单个项目可访问，可以创建一个本地 `.env` 文件，其中包含 API 密钥，然后使用 Python 代码显式使用该 API 密钥。

首先，导航到要创建 `.env` 文件的项目文件夹。然后，使用终端或集成开发环境（IDE）创建一个名为 `.env` 的文件。

请注意，为了防止您的 `.env` 文件被版本控制系统无意中分享，请在项目根目录中创建一个 `.gitignore` 文件，并在其中添加一行 `.env`。

一旦创建了 `.env` 文件，请将 API 密钥设置为 `OPENAI_API_KEY`：

```makefile
# 将您的 API 密钥粘贴在此处。请勿与任何人分享！API 密钥应保持私密。
OPENAI_API_KEY=abc123
```

然后，可以使用以下 Python 代码导入 API 密钥：

```python
from openai import OpenAI

client = OpenAI()
```

默认情况下，`OpenAI` 类将使用 `os.environ.get("OPENAI_API_KEY")` 自动检索 API 密钥。如果您将密钥保存在不同的环境变量名称下，可以这样做：

```python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("CUSTOM_ENV_NAME"),
)
```

## 步骤 3：发送您的第一个 API 请求

### 创建请求

创建一个名为 `openai_test.py` 的文件，并在其中粘贴以下代码：

```python
from openai import OpenAI

client = OpenAI()

response = client.completions.create(
    engine="davinci",
    prompt="Write a Python function to calculate the factorial of a number",
    max_tokens=100,
    n=1,
    stop=None,
    temperature=0.7,
)

print(response.choices[0].text)
```

在此示例中，我们使用 `completions.create` 方法向 OpenAI API 发送请求。我们将 `engine` 参数设置为 `davinci`，这是 OpenAI 提供的最强大的模型之一。我们还设置了 `prompt` 参数，要求模型编写一个计算数字阶乘的 Python 函数。

`max_tokens` 参数控制生成的文本的最大长度，`n` 参数控制生成的结果数量，`stop` 参数用于停止生成的文本，`temperature` 参数控制生成的文本的随机性。

### 运行请求

要运行请求，请在终端/命令行中导航到包含 `openai_test.py` 文件的目录，然后运行：

```bash
python openai_test.py
```

如果一切顺利，您应该会看到模型生成的 Python 函数，用于计算数字的阶乘：

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
```
